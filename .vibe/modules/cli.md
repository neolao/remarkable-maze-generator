# Module: cli
**Role:** Command-line interface (Commander) generating mazes as PDF and sending them to reMarkable. The `generate` (width/height/seed/difficulty/type/solution/output), `send` (upload a local PDF, with interactive first-time pairing), and `generate-and-send` (combines both in one call, same generation options as `generate`) commands are implemented.
**Files:** `packages/cli/src/index.ts`, `packages/cli/src/generate.ts`, `packages/cli/src/generate-and-send.ts`, `packages/cli/src/cli-options.ts`, `packages/cli/src/send.ts`, `packages/cli/src/credential-store.ts`
**Exports:** `remarkable-maze` binary (entry point `program.parse()`, subcommands `generate`, `send`, and `generate-and-send`; `generate` and `generate-and-send` support `--difficulty <number>` (1–5, see ADR 015), `--type <type>` (`rectangle` or `rectangle-crossing`, defaults to `rectangle`, validated via `core`'s shared `isValidMazeType`/`invalidMazeTypeMessage` — see ADR 022), and `--solution <mode>` (`none`, `extra-page`, or `overlay`, defaults to `none`, validated via `core`'s shared `isValidSolutionMode`/`invalidSolutionModeMessage` — see ADR 021); `send` and `generate-and-send` support `--folder <name>`), `runGenerate(options: GenerateOptions): Promise<GenerateResult>` (`options.solution` and `options.type` are raw strings validated internally, rejecting an unsupported value before generation starts), `parseIntegerOption(value, flagName): number`, `runSend(options: SendOptions): Promise<SendResult>` (prompts interactively for a pairing code on first use, per ADR 011), `runGenerateAndSend(options: GenerateAndSendOptions): Promise<GenerateAndSendResult>` (defaults the visible name to `rectangle-<width>x<height>-<seed>` per ADR 014; forwards `options.solution` and `options.type` to generation, rejecting either before any authentication or upload attempt; throws `UploadAfterGenerateError` — keeping the local PDF — if generation succeeds but upload fails), `createFileCredentialStore(filePath): CredentialStore` (JSON file at `~/.config/remarkable-maze-generator/credentials.json` by default, mode `0600`)
**Depends on:** [`modules/core.md`](core.md)
