# PDF Solution Display Option

## Description
Add an option to the PDF renderer to include the maze's solution, either as a separate page after the maze or as an overlay on the same page, selectable by the caller.

## Acceptance Criteria
- [ ] System can render a PDF with the solution as an extra page when the option is enabled
- [ ] System can render a PDF with the solution overlaid on the maze page when that mode is selected
- [ ] By default (option not set), the PDF contains no solution

## Notes
None.
